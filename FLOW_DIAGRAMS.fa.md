# 📊 نمودار جریان سیستم Leitner Box

## 🔄 نمودار کلی

```
┌─────────────────────────────────────────────────────────┐
│                    صفحه خانه                            │
│  کارت جدید: 10 | آماده: N | مجموع: 10+N             │
│  جعبه 1: 10 | جعبه 2: N | ... | جعبه 5: M            │
└─────────────────────────────────────────────────────────┘
                          ↓
                    [شروع مرور]
                          ↓
┌─────────────────────────────────────────────────────────┐
│                    صفحه مرور                           │
│  ۱. کارت‌های آماده (Box 2-5)                       │
│  ۲. کارت‌های جدید (Box 1)                          │
│                                                         │
│  [کارت]        [برای خوانندگی]                      │
│  ┌─────────────────────────┐                          │
│  │ English Term            │                          │
│  └─────────────────────────┘                          │
│     [فیلپ برای دیدن جواب]                            │
│  ┌─────────────────────────┐                          │
│  │ Persian Translation     │                          │
│  │ Pronunciation           │                          │
│  │ Example + Translation   │                          │
│  └─────────────────────────┘                          │
│  [❌ غلط] [😐 سخت] [✅ صحیح]                          │
└─────────────────────────────────────────────────────────┘
```

## 🏗️ معماری داده

```
LocalStorage
    │
    └─ 'ielts-collocation-leitner-state-v2'
       │
       ├─ categoryStates
       │  │
       │  ├─ 'اقتصاد'
       │  │  ├─ cards
       │  │  │  ├─ 1: { id, category, box, createdAt, nextReviewDate, ... }
       │  │  │  ├─ 2: { ... }
       │  │  │  └─ ...
       │  │  ├─ introducedCount: 50
       │  │  └─ lastIntroducedDate: '2024-01-15'
       │  │
       │  ├─ 'محیط‌زیست'
       │  │  └─ cards: { ... }
       │  │
       │  └─ ...
       │
       ├─ selectedCategory: 'اقتصاد'
       ├─ reviewHistory: [...]
       ├─ streak: 5
       ├─ lastStreakDate: '2024-01-15'
       └─ locale: 'fa'
```

## 🎬 جریان مرور (Flow)

```
شروع مرور
    ↓
حالت دسته = categoryStates['اقتصاد']
    ↓
کارت‌های آماده = filter(nextReviewDate <= امروز && createdAt != امروز)
کارت‌های جدید = filter(createdAt == امروز && box == 1)
    ↓
صف مرور = [...کارت‌های آماده, ...کارت‌های جدید]
    ↓
حلقه مرور:
    ├─ کارت کنونی = صف[indexفعلی]
    ├─ نمایش
    ├─ کاربر پاسخ می‌دهد
    │
    ├─ اگر ✅:
    │  ├─ box = box + 1
    │  ├─ reviewCount++
    │  └─ nextReviewDate = امروز + interval[box]
    │
    ├─ اگر 😐:
    │  ├─ box = box (بدون تغییر)
    │  ├─ reviewCount++
    │  └─ nextReviewDate = امروز + interval[box]/2
    │
    └─ اگر ❌:
       ├─ box = 1
       ├─ reviewCount++
       └─ nextReviewDate = امروز (فوری)
    ↓
ذخیره در LocalStorage
    ↓
کارت بعدی یا پایان مرور
```

## 📈 نمودار انتقال Box

```
                  صحیح                صحیح                صحیح
             ┌──────────┐         ┌──────────┐         ┌──────────┐
             ↓          │         ↓          │         ↓          │
        ┌────────────────────────────────────────────────────────────┐
        │ Box 1 (0d) → Box 2 (1d) → Box 3 (3d) → Box 4 (7d) → Box 5 (14d)
        │    ↓              ↓             ↓             ↓       
        │ غلط/سخت      غلط/سخت      غلط/سخت      غلط/سخت
        │    │              │             │             │        
        └────────────────────────────────────────────────────────────┘
             └──────────────────────────────────────┘
                    بازگشت به Box 1 (غلط)
```

## 📊 جدول انتقال Box

```
┌──────────┬────────────┬──────────────┬────────────────────┐
│ پاسخ     │ Box فعلی   │ Box جدید     │ nextReviewDate     │
├──────────┼────────────┼──────────────┼────────────────────┤
│ ✅ صحیح  │ 1          │ 2            │ امروز + 1          │
│ ✅ صحیح  │ 2          │ 3            │ امروز + 3          │
│ ✅ صحیح  │ 3          │ 4            │ امروز + 7          │
│ ✅ صحیح  │ 4          │ 5            │ امروز + 14         │
│ ✅ صحیح  │ 5          │ 5            │ امروز + 14         │
├──────────┼────────────┼──────────────┼────────────────────┤
│ 😐 سخت   │ 1          │ 1            │ امروز + 0 (نیمی)   │
│ 😐 سخت   │ 2          │ 2            │ امروز + 0.5        │
│ 😐 سخت   │ 3          │ 3            │ امروز + 1.5        │
│ 😐 سخت   │ 4          │ 4            │ امروز + 3.5        │
│ 😐 سخت   │ 5          │ 5            │ امروز + 7          │
├──────────┼────────────┼──────────────┼────────────────────┤
│ ❌ غلط   │ N          │ 1            │ امروز (فوری)       │
└──────────┴────────────┴──────────────┴────────────────────┘
```

## 🔍 نقاط مهم در نمودار

### ✅ موارد صحیح:
- ✓ کارت جدید همیشه در Box 1 شروع می‌شود
- ✓ nextReviewDate تعیین می‌کند چه زمانی ظاهر می‌شود
- ✓ کارت قبلی روز بعد یا بعد به ترتیب فاصله
- ✓ غلط شدن بازمی‌گرداند Box 1
- ✓ سخت بودن نیمی فاصله می‌دهد

### ❌ اشتباهات قدیمی:
- ✗ Box 1 روزانه 10+ داشت (حالا دقیق 10)
- ✗ کارت روز قبل ظاهر نمی‌شد (حالا ظاهر می‌شود)
- ✗ دسته‌ها با هم تأثیر داشتند (حالا مستقل)
- ✗ فواصل اشتباه بود (حالا 0,1,3,7,14)

## 🎯 خلاصه

```
Input:  کارت + پاسخ + تاریخ
  ↓
Logic:  قوانین Leitner Box
  ↓
Output: Box جدید + nextReviewDate
  ↓
Storage: LocalStorage
  ↓
View:   صفحه‌ای آپدیت شده
```

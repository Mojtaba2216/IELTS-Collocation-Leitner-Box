# 📅 جدول زمان‌بندی جعبه لایتنر

## 🎯 الگوریتم Leitner Box اصلاح شده

### فاصله مراجعه برای هر جعبه:

```
┌─────────────┬──────────────┬────────────────────────────────┐
│ جعبه        │ فاصله        │ توضیح                          │
├─────────────┼──────────────┼────────────────────────────────┤
│ Box 1       │ 0 روز        │ مراجعه فوری (همان روز)        │
│ Box 2       │ 1 روز        │ روز بعد                       │
│ Box 3       │ 3 روز        │ سه روز بعد                    │
│ Box 4       │ 7 روز        │ یک هفته بعد                   │
│ Box 5       │ 14 روز       │ دو هفته بعد (تسلط کامل)      │
└─────────────┴──────────────┴────────────────────────────────┘
```

## 🔄 نمودار جریان تصمیم

```
کارت در جعبه N
        ↓
مراجعه امروز
        ↓
      ╭─────────────────────────┬──────────────────┬─────────────────────╮
      ↓                        ↓                   ↓                     ↓
   ❌ غلط               😐 سخت                ✅ صحیح           ✅ صحیح (Box 5)
      │                        │                   │                     │
      └─→ Box 1           └─→ Box N           └─→ Box N+1            └─→ Box 5
          nextReview:      nextReview:         nextReview:           nextReview:
          امروز            امروز + نصف         امروز +               امروز +
                          فاصله             فاصلة کامل              14 روز
```

## 📈 چگونگی پیشرفت

### صحیح پاسخ دادن (مسیر سریع):
```
Day 0:  [Box 1] 10 cards
Day 1:  [Box 2] 5 cards → [Box 1] 10 new + 5 remaining
Day 4:  [Box 3] 3 cards → [Box 2] 5 cards + [Box 1] 10 new
Day 11: [Box 4] 2 cards → [Box 3] 3 cards + [Box 2] 5 cards + [Box 1] 10 new
Day 18: [Box 5] 1 card  → [Box 4] 2 cards + ... + [Box 1] 10 new
       (تسلط کامل)
```

### اگر اشتباه بشود:
```
Day 2:  [Box 2] 5 cards
Day 2:  ❌ اشتباه
        └─→ [Box 1] بازمی‌گردد
        │
Day 2:  10 + 1 (اشتباه شده) = 11 کارت
Day 3:  کارت اشتباه شده دوباره در Box 1
```

## 🧮 حسابات مختلف

### حالت ایده‌آل (همه صحیح):
```
اگر 10 کارتی برای روز 1 اضافه شود و تمام صحیح باشد:

Day 1:   Box 1 = 10
Day 2:   Box 1 = 10 (new), Box 2 = 10 (from day 1)
Day 3:   Box 1 = 10 (new), Box 2 = 10 (new ready)
Day 4:   Box 1 = 10 (new), Box 2 = 10, Box 3 = 10 (from day 1)
Day 5:   Box 1 = 10 (new), Box 2 = 10, Box 3 = 10, Box 4 = 10 (ready)
...
Day 18:  Box 1 = 10 (new), Box 2, Box 3, Box 4, Box 5 = 10 (from day 1)
```

### حالت واقعی (مخلوط):
```
50% دقت:
Day 1:  Box 1 = 10
Day 2:  Box 1 = 10 (new) + 5 (wrong) = 15
        Box 2 = 5 (correct)
Day 3:  Box 1 = 10 (new) + 3 (wrong) = 13
        Box 2 = 7 (ready)
        Box 3 = 2 (promoted)
```

## ⚡ سرعت‌های مختلف

### سریع (همه صحیح):
```
Box 1 → Box 2 → Box 3 → Box 4 → Box 5
0d      1d      3d      7d      14d
↑───────────────────────────────↑
روز 11 به تسلط کامل می‌رسید
```

### متوسط (70% صحیح):
```
Box 1 → Box 2 ←─────┐
↓                   │
(wrong) ────────────┘
روز 14-20 به تسلط
```

### آهسته (50% صحیح):
```
Box 1 ↔ Box 2 ↔ Box 3
↓     ↓     ↓
(wrong) دوباره شروع
روز 25+ به تسلط
```

## 🎯 نقاط کلیدی

### ✅ درست:
```javascript
// Box 1 کارت
nextReviewDate = today           // مراجعه فوری
lastReviewedDate = ''            // هنوز مراجعه نشده

// پس از صحیح جواب
box = 2                          // رفتن به Box 2
nextReviewDate = today + 1d      // فردا
lastReviewedDate = now           // زمان مراجعه

// پس از اشتباه جواب
box = 1                          // برگشت به Box 1
nextReviewDate = today           // امروز دوباره
reviewCount = reviewCount + 1    // شمارش مراجعات
```

### ❌ غلط:
```javascript
nextReviewDate = '2024-12-25'    // ❌ تاریخ سخت کد
reviewCount = 0 + 1              // ❌ عملیات اشتباه
nextReviewDate < createdAt       // ❌ منطقی غیرممکن
box > 5                          // ❌ Box بالاتر نیست
```

## 🧪 اختبار سریع

```javascript
// در Console:

// ۱. دسته امروز
const today = new Date().toISOString().split('T')[0];
const data = JSON.parse(localStorage.getItem('ielts-collocation-leitner-state-v2'));

// ۲. شمارش جعبه ۱
const box1Today = Object.values(data.categoryStates['اقتصاد'].cards)
  .filter(c => c.box === 1 && c.createdAt === today)
  .length;
console.log(`جعبه 1 امروز: ${box1Today}`); // باید 10 باشد

// ۳. کارت‌های آماده مرور
const readyToReview = Object.values(data.categoryStates['اقتصاد'].cards)
  .filter(c => c.nextReviewDate <= today && c.createdAt !== today)
  .length;
console.log(`آماده مرور: ${readyToReview}`);

// ۴. Box 2 کارت‌های فردا
const tomorrow = new Date();
tomorrow.setDate(tomorrow.getDate() + 1);
const tomorrowStr = tomorrow.toISOString().split('T')[0];
const box2Tomorrow = Object.values(data.categoryStates['اقتصاد'].cards)
  .filter(c => c.box === 2 && c.nextReviewDate === tomorrowStr)
  .length;
console.log(`جعبه 2 فردا: ${box2Tomorrow}`);
```

# Leitner Box System - تحديثات منطق النظام

## ✅ مشاكل المصلحة:

### 1. **توليد البطاقات اليومي**
- ✅ تم الإصلاح: كل فئة تحصل على **10 بطاقات جديدة فقط يومياً** من Box 1
- ✅ البطاقات لا تُكرر - يتم تتبع الكروت المُقدَّمة بالفعل
- ✅ كل فئة لها حالة مستقلة تماماً

### 2. **جدولة مراجعة Box**
تم تصحيح فترات المراجعة:

```
Box 1: مراجعة يومية (نفس اليوم)
       ↓ عند الإجابة الصحيحة
Box 2: بعد 1 يوم
       ↓
Box 3: بعد 3 أيام
       ↓
Box 4: بعد 7 أيام
       ↓
Box 5: بعد 14 يوم (إتقان كامل)
```

### 3. **آلية الإجابة على البطاقات**

#### ❌ إجابة خاطئة (Wrong):
```
Box N → Box 1 (العودة للبداية)
nextReviewDate = اليوم (مراجعة فوراً)
```

#### 😐 إجابة صعبة (Hard):
```
Box N → Box N (ابقاء في نفس الصندوق)
nextReviewDate = اليوم + (نصف الفترة)
مثال: في Box 2 (فترة 1 يوم) = اليوم + 0.5 أيام
```

#### ✅ إجابة صحيحة (Correct):
```
Box 1 → Box 2 (nextReviewDate = اليوم + 1)
Box 2 → Box 3 (nextReviewDate = اليوم + 3)
Box 3 → Box 4 (nextReviewDate = اليوم + 7)
Box 4 → Box 5 (nextReviewDate = اليوم + 14)
Box 5 → Box 5 (ثابت، مراجعة كل 14 يوم)
```

### 4. **البيانات المخزنة لكل بطاقة**

```typescript
{
  id: number;              // معرف البطاقة
  category: string;        // اسم الفئة (مثل "اقتصاد")
  box: 1 | 2 | 3 | 4 | 5; // الصندوق الحالي
  createdAt: string;       // تاريخ إضافة البطاقة (YYYY-MM-DD)
  nextReviewDate: string;  // تاريخ المراجعة التالية
  lastReviewedDate: string;// تاريخ آخر مراجعة
  reviewCount: number;     // عدد مرات المراجعة
}
```

### 5. **عرض الفئة (Category Page)**

تُعرض المعلومات التالية:

```

🆕 كارت‌های جدید امروز: 10

✅ آماده مرور امروز: [عدد]
   (بطاقات من الأيام السابقة جاهزة للمراجعة اليوم)

📦 جعبه 1: 10  (بطاقات جديدة اليوم)
📦 جعبه 2: [عدد]  (جاهز للمراجعة غداً)
📦 جعبه 3: [عدد]  (جاهز خلال 3 أيام)
📦 جعبه 4: [عدد]  (جاهز خلال 7 أيام)
📦 جعبه 5: [عدد]  (مُتقن، جاهز خلال 14 يوم)
```

### 6. **تتابع المراجعة اليومي**

```
يوم 1 (الإثنين):
├─ 10 بطاقات جديدة من Box 1
├─ مراجعة المستخدم
└─ (صحيح) → Box 2 (المراجعة غداً)
   (خطأ) → Box 1 (المراجعة اليوم)

يوم 2 (الثلاثاء):
├─ 10 بطاقات جديدة من Box 1 ✨
├─ بطاقات من Box 2 جاهزة (من الأمس)
└─ اختياري: مراجعة Box 1 مرة أخرى

يوم 3 (الأربعاء):
├─ 10 بطاقات جديدة من Box 1 ✨
├─ بطاقات من Box 2 جاهزة
└─ بطاقات من Box 3 جاهزة (من قبل يومين)
```

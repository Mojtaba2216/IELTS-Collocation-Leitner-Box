# سیستم Leitner Box - راهنمای سریع

## 🎯 نقاط کلیدی

### ✅ مشکلات حل شده:

1. **۱۰ کارت روزانه** - اکنون دقیقاً ۱۰ کارت جدید برای هر فئه روزانه
2. **جعبه های مستقل** - هر فئه حالت خودش را دارد
3. **برنامه صحیح** - بطاقات در روزهای درست ظاهر می شوند
4. **ذخیره سازی** - تمام داده ها در localStorage محفوظ می شوند
5. **تست آسان** - `simulateNextDay()` برای آزمایش

---

## 📊 درخت تصمیم Box

```
عضو پاسخ
    ├─ ❌ (غلط)     → Box 1 (امروز)
    ├─ 😐 (سخت)     → همان Box (نیمی از فاصله)
    └─ ✅ (صحیح)   → Box بعدی (فاصله کامل)
```

Box 1: 0 روز  (همان روز)

## ✨ مثال عملی

**روز ۱ (دوشنبه):**
```
صبح:
├─ ۱۰ کارت جدید → Box 1
└─ کل: ۱۰ کارت

بعد مرور:
├─ 5 کارت: صحیح → Box 2 (مراجعه فردا)
├─ 3 کارت: سخت → Box 1 (فردا ۶۰ درصد فاصله)
└─ 2 کارت: غلط → Box 1 (امروز)
```

**روز ۲ (سه‌شنبه):**
```
صبح:
├─ ۱۰ کارت جدید → Box 1 ✨
├─ 5 کارت از Box 2 (از دیروز)
└─ کل: ۱۵ کارت

نوبت مرور:
کارت های Box 2 اول
سپس کارت های جدید
```

**روز ۳ (چهار‌شنبه):**
```
صبح:
├─ ۱۰ کارت جدید → Box 1 ✨
├─ کارت های Box 2
├─ کارت های Box 3 (اگر از روز ۱ صحیح پاسخ دادند)
└─ کل: ۱۸+ کارت
```

## 📞 هنگام مشکل

1. **بطاقات ظاهر نمی‌شوند؟**
   - چک کنید: `nextReviewDate <= today`

2. **یک فئه داده های دیگر را تأثیر می‌دهد؟**
   - بررسی کنید: هر فئه فئه خودش است

3. **localStorage خالی است؟**
   - صفحه را refresh کنید
   - یا: `loadProgress()` را دوبار فراخوانی کنید

4. **تاریخ اشتباه؟**
   - استفاده کنید: `getTodayString()`
